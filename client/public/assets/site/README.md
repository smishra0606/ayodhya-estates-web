# Site Photos

Place your Panchi Vihar site development photos here.

These images can be used for:
- Static sections of the website
- Background overlays
- About section imagery

**Note:** The dynamic gallery on the homepage pulls images from the Admin Panel (Cloudinary), NOT from this folder.

This folder is for:
- Hardcoded/static images in the design
- Background imagery
- Branding elements

To use images from this folder in components:
```javascript
<img src="/assets/site/your-image.jpg" alt="Description" />
```
