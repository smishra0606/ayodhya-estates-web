# Hero Section Images

Place your Lord Ram banner and hero background images here.

Current hero image is loaded from URL in Hero.css:
```
background: url('https://i.ibb.co/Lpfpz5M/lord-rama-sita.jpg')
```

To use a local image instead:
1. Add your image file here (e.g., `lord-rama-sita.jpg`)
2. Update `client/src/components/Hero.css` line with:
   ```
   background: url('/assets/hero/lord-rama-sita.jpg')
   ```

Recommended specs:
- Format: JPEG or WebP
- Dimensions: 1920x1080px or higher
- File size: < 500KB
- Aspect ratio: 16:9
